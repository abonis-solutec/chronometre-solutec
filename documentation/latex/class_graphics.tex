\doxysubsection{Detailed Description}
Classe gérant tout l\textquotesingle{}affichage graphique. 

Cette classe permet de créer une fenêtre graphique en utilisant la librairie Swing. Elle gère également la récupération des valeurs du temps devant être affiché.
